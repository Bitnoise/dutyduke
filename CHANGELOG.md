# Changelog

All notable changes to this project will be documented in this file.

This project uses [Conventional Commits](https://www.conventionalcommits.org/) and follows [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-03-06

### Initial Release

- Employee management with profiles, onboarding, employment history, skills, and education
- Absence management with leave requests, approval workflows, availability calendar, and iCal export
- Document management with upload, categorization, expiration tracking, and assignment
- Equipment tracking with inventory, assignments, changelog, and status management
- Benefits administration with plan creation and employee assignment
- Performance feedback scheduling and tracking
- Role-based access control with granular per-resource permissions
- CV generation with PDF export
- Multi-language support (English, Polish) via next-intl
- Field-level encryption for sensitive data
- Email notifications for account invitations and password resets
- Clean Architecture with Repository and Use Case patterns
- Docker Compose setup with PostgreSQL (PostGIS) and Mailcatcher
